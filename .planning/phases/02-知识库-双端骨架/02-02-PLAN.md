- `embed(text)`: POST to `${baseUrl}/embeddings` with body `{ model, input: text }`, OpenAI-compatible format

  - Response: `{ data: [{ embedding }, ...] }` — map all embeddings
- `dimension`: detect from first successful call, cache and return
- Error handling: if fetch fails, throw with clear message including URL and model name

- Constructor takes { apiKey: string, model?: string } (model defaults to "text-embedding-3-small")

4. Create factory function:
```typescript
export function createEmbeddingProvider(config?: {
  provider?: "ollama" | "openai" | "mock";
  baseUrl?: string;
  model?: string;
  apiKey?: string;
}): EmbeddingProvider {
  const provider = config?.provider || process.env.EMBEDDING_PROVIDER || "ollama";
  if (provider === "ollama") {
    return new OllamaEmbeddingProvider({
      baseUrl: config?.baseUrl || process.env.EMBEDDING_BASE_URL || "http://localhost:11434/v1",
      model: config?.model || process.env.EMBEDDING_MODEL || "nomic-embed-text-v2-moe",
    });
  }
  if (provider === "openai") {
    return new OpenAIEmbeddingProvider({
      apiKey: config?.apiKey || process.env.OPENAI_API_KEY || "",
      model: config?.model || "text-embedding-3-small",
    });
  }
  if (provider === "mock") {
    return new MockEmbeddingProvider(); // returns deterministic 768-d vectors for testing
  }
  throw new Error(`Unknown embedding provider: ${provider}`);
}
```

```typescript
  insert(table: string, records: Record<string, unknown>[]): Promise<void>;
  getAll(table: string, options?: { limit?: number; offset?: number; filter?: string }): Promise<KnowledgeResult[]>;
  getById(table: string, id: string): Promise<KnowledgeResult | null>;

  delete(table: string, id: string): Promise<void>;
  count(table: string): Promise<number>;
}
```

3. Create LanceDBFileStore class:
- Constructor: `{ dbPath: string, embeddingProvider: EmbeddingProvider }`
- Private method `getOrConnectTable(table: string, schema: ArrowSchema)`:
  - Connect to DB: `await connect(dbPath)`
  - Check if table exists, create if not with schema
  - Return table reference
- `search(table, query, options)`:
  - If query is string: call `embeddingProvider.embed(query)` to get vector, then `table.search(vector).limit(options?.limit || 10).toArray()`
  - If query is number[]: direct vector search
  - Apply filter via `.where(options?.filter)` if provided
  - Return results with id and score (LanceDB distance)
- `insert(table, records)`: `table.add(records)`
- `getAll(table, options)`: `table.toArrow()` or `.filter()`, convert to JS objects, apply pagination
- `getById(table, id)`: search with filter `id = '${id}'`
- `update(table, id, data)`: LanceDB doesn't support update — delete + re-insert
- `delete(table, id)`: `table.delete(\`id = '${id}'\`)`
- `count(table)`: return array length from getAll
