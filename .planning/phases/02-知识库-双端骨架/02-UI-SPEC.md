# Phase 02 — UI Design Contract

> Visual and interaction contract for unified frontend skeleton (student + admin in NextJS 15 + shadcn/ui). Generated by wsf-ui-researcher, verified by wsf-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn/ui |
| Preset | default (CSS variables) |
| Component library | Radix UI (via shadcn) |
| Icon library | Lucide Icons (standard), student theme may add emoji |
| Font | Inter (student theme), system stack (admin theme) |

**Dual-theme strategy:** Single shadcn component library with two CSS variable sets. Student theme uses warm colors, rounded-xl, larger fonts. Admin theme uses neutral colors, rounded-md, compact density.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: 
- Icon-only touch targets: 44px minimum (WCAG compliance)
- Student card padding: 20px (slightly larger for breathing room)

## Color

### Student Theme

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFBF5 (warm white) | Background, surfaces |
| Secondary (30%) | #FFF8E7 (warm gray) | Cards, sidebar, nav background |
| Accent (10%) | #F59E0B (amber-500) | Primary buttons, active nav, focus indicators |
| Accent-secondary | #14B8A6 (teal-500) | Success states, progress indicators |
| Destructive | #EF4444 (red-500) | Delete buttons, error states |

Accent reserved for: Primary CTA buttons, active navigation items, focus ring, loading spinners. Accent-secondary for success badges, progress bars.

### Admin Theme

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F8FAFC (slate-50) | Background, surfaces |
| Secondary (30%) | #F1F5F9 (slate-100) | Cards, sidebar, nav background |
| Accent (10%) | #3B82F6 (blue-500) | Primary buttons, links, active nav |
| Accent-secondary | #6366F1 (indigo-500) | Hover states, secondary actions |
| Destructive | #DC2626 (red-600) | Delete buttons, destructive confirmation |

Accent reserved for: Primary buttons, navigation active state, table row selection, link hover. Accent-secondary for secondary button hover, icon hover.
