| polled progress payload → FEEDBACK UI | 前端根据 progress 中的评分结果切换状态与导航 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-13-01 | Tampering | `/answer-score` counter update | mitigate | 仅在 answer row 尚未计入时更新聚合计数，避免重复 tool 调用造成 double-count。 |
| T-03-13-02 | Repudiation | latest feedback polling | mitigate | `latest_feedback` 绑定最近 answer row 的 question_id/score/feedback，避免前端拿错旧反馈。 |
| T-03-13-03 | Denial of Service | FEEDBACK gating | accept | 轮询继续使用已有 `/progress` 端点，状态机需容忍重复 done / assessment_done 信号。 |
</threat_model>
