**Logic:**
1. Count existing questions in SQLite. If count >= 16, skip (idempotent).
2. Insert 16 questions into SQLite `assessment_questions` table with status='active'.
3. For each question, build embedding text: `${content} ${knowledge_point} ${explanation || ''}`.
4. Embed all questions in batch via `embedder.embedBatch(texts)`.
5. Insert into LanceDB `assessment_questions` table with `vector` field.

**Insert into SQLite:**
```typescript
  const existingCount = await db.$count(assessmentQuestions);
  if (existingCount >= QUESTIONS.length) {
    logger.info({ existing_count: existingCount }, "Assessment questions already seeded, skipping");
    return;
  }

  // Insert into SQLite
  const inserted = await db.insert(assessmentQuestions)
    .values(QUESTIONS.map(q => ({
      ...q,
      id: crypto.randomUUID(),
      created_at: new Date(),
      updated_at: new Date(),
    })))
    .returning();

  logger.info({ count: inserted.length }, "Assessment questions inserted into SQLite");
```

**Insert into LanceDB:**
```typescript
  // Prepare LanceDB records with embeddings
  const texts = QUESTIONS.map(q =>
    `${q.content} ${q.knowledge_point} ${q.explanation || ''}`
  );
  const embeddings = await embedder.embedBatch(texts);

  const lanceRecords = inserted.map((row, i) => ({
    id: row.id,
    course_id: row.course_id,
    level: row.level,
    knowledge_point: row.knowledge_point,
    question_type: row.question_type,
    content: row.content,
    explanation: row.explanation || '',
    vector: embeddings[i],
  }));

  // Insert into LanceDB (will create table if not exists)
  await vectorStore.insert("assessment_questions", lanceRecords);
  logger.info({ count: lanceRecords.length }, "Assessment questions indexed in LanceDB");
```

**Idempotency check:** The function checks `db.$count(assessmentQuestions)` before inserting. If count >= expected, skip all operations. This allows the seed to be called on every startup without duplicating data.
